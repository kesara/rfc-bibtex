# rfc-bibtex
Compile a BibTeX file with RFC bibliographic references from Datatracker.

## Setup environment
```
python3 -m venv venv
. venv/bin/activate
pip install -r requitements.txt
```

## Generate BibTeX file
```
python generate-bibtex.py
```

## Validate BibTeX file
```
python validate-bibtex.py
```
